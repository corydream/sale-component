SelectBox 平铺树选择

平铺树选择控件

何时使用

类似 Select 的选择控件，可选择的数据结构是一个平铺的树形结构时，可以使用 SelectBox，例如公司层级、部门人员展示及选择等等

支持拼音搜索，首字母搜索

SelectBox组件主要用于ERP中，用于平铺展示部门数据及选择。数组中的每个对象只能有一个子集！！！

代码演示

选择框最大显示3个item，树选择排列4个项一列，展示框高度120px

不带搜索功能的

设置bgxIsOpen，展开所有树选择项

设置span和maxDisplayed进行改变选择框显示个数，及树选择排列个数

三种大小的选择框，当 bgxSize 分别为 large 和 small 时，输入框高度为 40px 和 24px ，默认高度为 32px

设置bgxOnlyShowParent=true，即所有子节点选中时显示父节点
