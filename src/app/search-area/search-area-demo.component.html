SearchArea 搜索区域

基于OA业务场景，动态创建搜索区域

何时使用

需要用户处理事务，通过配置快速生成搜索区域，承载相应的操作。

搜索区域项过多时，可进行拖拽排序或筛选所需条件、持久化用户搜索习惯

朱林峰   zhulinfeng@banggood.com

代码演示

支持添加前置/后置标签、前缀图标

支持选择搜索 serverSearch 为 true ，可根据 onSearch 回调请求异步接口获取数据设置列表项

type='nz' 还支持 scrollToBottom 回调下拉列表滚动到底部进行动态加载数据

支持树选择及平铺树选择

可通过 Observable 处理异步请求数据， next 返回的数组对象

支持日期选择框，日期范围选择框，星期选择框

支持 Checkbox 组

在基础控件不满足的情况下，开发者可自定义模板来处理业务场景

结合 保存常用查询 功能进行综合使用

可设置每项的 order 进行排序展示

搜索区域超出宽度自动换行并隐藏，可点击更多按钮显示隐藏区域，可进行拖拽排序及筛选条件，持久化用户搜索习惯

bgxOnSearch 返回搜索内容 {{ '{ [key: string]: any｜any[]; }' }}
